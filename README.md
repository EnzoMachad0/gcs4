# TaskManager — Gerenciador de Tarefas

Aplicação simples em Python para criação e acompanhamento de tarefas com prioridades.
Desenvolvida como projeto de exemplo para a disciplina de **Gerência de Configuração de Software (GCS)**.

---

## Estrutura de Diretórios

```
gcs-projeto/
├── src/              # Código-fonte principal
│   └── task_manager.py
├── tests/            # Testes automatizados
│   └── test_task_manager.py
├── config/           # Módulo de configuração
│   └── settings.py
├── docs/             # Documentação adicional
├── scripts/          # Scripts utilitários
│   └── run.sh
├── config.env        # Variáveis de ambiente (template)
├── CONFIG_MAP.md     # Mapa de Itens de Configuração
└── README.md         # Este arquivo
```

---

## Pré-requisitos

| Dependência | Versão mínima |
| ----------- | ------------- |
| Python      | 3.11          |

Não há dependências externas — a aplicação usa apenas a biblioteca padrão do Python.

---

## Instalação e Execução

```bash
# 1. Clone o repositório
git clone git git@github.com:EnzoMachad0/gcs4.git
cd gcs-projeto

# 2. (Opcional) Ajuste as configurações
cp config.env .env
# edite .env conforme necessário

# 3. Execute a aplicação
bash scripts/run.sh
# ou diretamente:
python src/task_manager.py
```

---

## Testes

```bash
python tests/test_task_manager.py
```
